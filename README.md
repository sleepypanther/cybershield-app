# 🛡️ CyberShield

CyberShield is a cross-platform cybersecurity dashboard that provides real-time system monitoring, vulnerability detection, and network analysis.

## 🚀 Features

* 🔍 System Scan
* 🌐 Network Monitoring
* 📂 File Risk Analyzer
* ⚠️ Vulnerability Detection
* 🔓 Breach Detection
* 🔧 Auto Fix System

## 🖥️ Tech Stack

* Frontend: React + Vite
* Backend: Node.js

## ⚙️ Setup

### Backend

cd backend
npm install
npm start

### Frontend

cd frontend
npm install
npm run dev

## 📦 Build

cd frontend
npm run build

## 📌 Note

node_modules is not included. Run `npm install` before starting.
